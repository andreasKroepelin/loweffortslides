# 1

How to use

Modify this template file (get it from GitHub): 

$ git clone https://github.com/andreasKroepelin/loweffortslides.git

One <section> per slide: 

<section id="42">
    <h2> slide title </h2>
    <p> some content </p>
    <div> # 42 </div>
</section>

Last <div> can contain slide number.

# 2

How to navigate

  • next slide: press PgDn or →, or scroll down
  • previous slide: press PgUp or ←, or scroll up
  • set anchor part in URL to slide number (…/…/file.html#42)

# 4

Behind the scenes: JS

Even less JavaScript: 

var idx = parseInt(location.href.split("#").pop());
if (isNaN(idx)) { idx = 1; }
function next() {
    idx += 1;
    if (document.getElementById("" + idx) == null) { idx -= 1; }
    location.href = "#" + idx;
}
function prev() {
    if (idx > 1) { idx -= 1; }
    location.href = "#" + idx;
}
document.addEventListener("keydown", e => {
    if (e.key == "PageDown" || e.key == "ArrowRight") {
        next();
    } else if (e.key == "PageUp" || e.key == "ArrowLeft") {
        prev();
    }
});
document.addEventListener("wheel", e => {
    if (e.deltaY > 0) { next(); } else { prev(); }
});
